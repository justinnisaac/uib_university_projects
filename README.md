# Computer Science software projects
## Content preview
The 21722 - AI Folder contains 3 software projects related to Artificial Intelligente, a subject I took during the 2025-26 course.
### Project 1 / Práctica 01: Search / Búsqueda
The first project is used as an introduction to the subject. The objective is to implement 3 different algorithms (DF, A*, Minimax) and compare
their efficience using Robot dummies as agents.

### Project 2 / Práctica 02: RL
The aim of the second project is to visualize the effect of using different techniques/algorithms such as Dynamic Programming, Genetic Algorithms, Monte Carlo algorithm, SARSA and Q-Learning. To do so,
each algorithm is implemented and tested on an agent. 

### Project 3 / Práctica 03: ML
The objective of the final project is to compare the capacity of predicting and clasifying values of different ML models such as Random Forest, SVM, Logistic Regression, Perceptron and Decision Trees.
Each model is evaluated on a previously analized and preprocessed dataset.
