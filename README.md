## 21722 - AI Folder 
It contains 3 software projects related to Artificial Intelligente, a subject I took during the 2025-26 course.
### Project 1 / Práctica 01: Search / Búsqueda
The first project is used as an introduction to the subject. The objective is to implement 3 different algorithms (DF, A*, Minimax) and compare
their efficience using Robot dummies as agents.

### Project 2 / Práctica 02: RL
The aim of the second project is to visualize the effect of using different techniques/algorithms such as Dynamic Programming, Genetic Algorithms, Monte Carlo algorithm, SARSA and Q-Learning. To do so,
each algorithm is implemented and tested on an agent. 

### Project 3 / Práctica 03: ML
The objective of the final project is to compare the capacity of predicting and clasifying values of different ML models such as Random Forest, SVM, Logistic Regression, Perceptron and Decision Trees.
Each model is evaluated on a previously analized and preprocessed dataset. The whole analysis is documented on a Jupyter Notebook.

## 21726 - DDBB 2 Folder
This folder contains the final project of the Database II subject, which I took during the 2025-26 course.
It consists of the creation of a web that allows users and volunteers help complete tasks related to the maintenance of cat colonies in Mallorca. The web also includes different profile management, since users can be classified as Volunteers, Volunteer Group Leader, Town Hall member and Vets, each one with its unique UI and functionalities. The database to store and query all the necessary information is also created using PhpMyAdmin and MySQL as the DBMS chosen.
This project puts into practice different important DB concepts: transactions, procedures, triggers, queries, events...
Technologies used: PHP, HTML, CSS, SQL.
